MOST USED HOTKEYS:
---------------
Ctrl+A = ChannelBox/Attribute Editor
Space = Tottle Viewports

Alt+LMB = Rotate
Alt+MMB = Pan
Alt+RMP = Zoom

F = Frame Select

Q = Select
W = Translate/Move
E = Rotate
R = Scale Tool

G = Last Used Tool

Ctrl+D = Duplicate.
B = Soft Select
Hold B+LMB Drag = Soft Select size adjust
---------------
Misc
---------------
Alt+B = Change Backgrounds
Ctrl+B = Paints Effects Brush Settings
----
Snap
----
X = Snap to Grid
C = Snap to Curve
V = Snap to Point/Vertex
